G# Framework Installation

1.       Unzip the G#Framework zip-file.

2.       Run the G#Installer.vi in the installation zip-file.

3.       Press Install button.

4.       Restart LabVIEW.

5.       Done!

Files will be installed in the following places:

<LabVIEW>/examples/AddQ

·         Examples

<LabVIEW>/help/_AddQ

·         Online help

<LabVIEW>/menus/Categories/AddQ_G#

·         G# function palette menu.

<LabVIEW>/project/errors

·         G# error code definitions

<LabVIEW>/resource/Framework/Provides/AddQ

·         G# IDE (Integrated Development Environmen). LabVIEW project integration and new menus G# IDE will integrate in project environment.

<LabVIEW>/vi.lib/addons/_AddQ

·         G# - G#Object, G#Interface, G#Debugger and G#Templates

The only files that is needed when executing a LabVIEW G# application is the VIs in <LabVIEW>/vi.lib/addons/_AddQ. Make sure these are always installed (this is also called G# runtime).

 

Uninstallation

 

1.       Run the G#Installer.vi in the installation zip-file.

2.       Press Uninstall button.

3.       Restart LabVIEW.

4.       Done!
