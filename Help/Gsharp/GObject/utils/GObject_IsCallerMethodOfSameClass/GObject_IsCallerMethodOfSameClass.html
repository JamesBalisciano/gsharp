G#Object.lvclass:G#Object_IsCallerMethodOfSameClass.vi

Checks if caller method specified by the call depth is of same class as the run-time input reference.

 reference in

Object reference in.

call depth (1)

Sets which VI in the call chain to use as reference VI.

is of same class?

If true, the calling VI belongs to the same class as the run-time reference.

reference full class name

Full class name of the input object reference.

is caller of G#Object?

If true, the calling VI method is of the G#Object class.
