reference in

Object reference in.

create method

Specifies the way of creating the object. 1) Create - a new object is always returned. Object name is ignored. 2) Create or lookup existing - a new object is created if name is unique within the class, if an object with the same name exists, this object is returned. 3) Lookup existing - object is looked up by name, if not existing an error is returned.

user counter (T)

If true, user counter is used when an object with a name that already exists.

object name ("")

Name of the new object. Unique within the class.

no registration (F)

If false, no registration of the object occurs. It will not show up in the debugger, be affected by garbage collector or listed by any method.
