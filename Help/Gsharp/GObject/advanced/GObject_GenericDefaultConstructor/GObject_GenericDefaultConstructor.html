G#Object.lvclass:G#Object_GenericDefaultConstructor.vi

Creates an instance of the class specified by the path by running the "G#Object_Create.vi" of the class. Override this construcor, else the default constructor runs. If the external create process is running, an event will be sent to let this process create and own the VI. This way LabVIEW garbage collector will not clear the object until it is destroyed or the garbage collector runs.

Be careful when using this VI when building executables, since paths must be correct and make sure that all needed VIs are included in the build. Do not exclude unused dynamic dispatch VIs in the build script.

Note: Will not work on RT systems.
