G#Debugger GUI - Objects tab

This tab will show all objects. The objects are grouped together in each class. Select an object in the tree view to see information about the object state.

The information about each object is:

·         Object Name

·         Object ID

·         Owner (top-level VI that created the object)

·         Full Class Name (unique name space)

·         # of Users

·         Counter (if enable or not)

·         Valid Reference (is object alive)

·         To String (value of the overridden ToString method, if implemented).

The state of the object (attributes value) for each inherited class in class hierarchy is shown as the object attributes control in the subpanel window.

Tab Control

Selects between Objects, Trace or Active Debug mode.

 

Refresh

Refresh the object list.

 

Destroy

Destroys the selected object and runs the Destroy method of the class.

 

Connect

Connects to a remote target. Make sure G#DebugServer is running on the remote target.

 

Garbage collect

Garbage collects all objects of all classes.

 

ObjectTree

All objects that is currently alive. If objects been created with "no registration" option, they will not show up here.

 

Inheritance

Selects which attributes to watch from a inheritance chain.

 

Attributes

Selects between object or class attributes.

 

Poll time (ms)

Attributes polling time in ms.

 

Highlight Aggregates?

Highlights the aggregate objects in the object tree.

 

Hide Empty Classes?

Removes classes with no objects from the tree view. Notice that a class could still have valid class attributes.

 

External Create Process Running

Indicates if the background external create process is running.

 

Window Transparency

Transparency of the front panel window.

 

Window Behaviour

Selects if the debugger is on top of all VIs or not.

 

# of Users

Number of users for the selected object.

 

Counter

Indicates if counter is active or not for the selected object.

 

Ref Valid

Is the selected object valid?

 

Object Name

Name of the object (if named).

 

Full Class Name

Full class path including libraries.

 

To String

Shows the "To String" implementation of the selected object.

 

G# Version

Version of G#.

 

Object ID

Unique object identity.

 

Application Instance

Application instance of the object scope.

 

Owner

Owning top-level VI of the selected object. When this VI stops executing, LabVIEW will automatically garbage collect the object.
