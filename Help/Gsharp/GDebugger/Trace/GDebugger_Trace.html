G#Debugger GUI - Trace tab

This tab shows a time stamped trace of all G# events. It is also possible to add user defined events by using the G#Object_SendUserEvent.vi to send events. All these events will also show up in the NI LabVIEW Desktop Execution Trace toolkit (if installed).

Tab Control

Selects between Objects, Trace or Active Debug mode.

 

Save

Saves log to file.

 

Trace Direction

Selects if log posts is added in front or not of the log.

 

Max Lines

Maximum number of lines in the trace window.

 

Clear

Clears log window.

 

External Create Process Running

Indicates if the background external create process is running.

 

Window Transparency

Transparency of the front panel window.

 

Window Behaviour

Selects if the debugger is on top of all VIs or not.

 

Trace

Time log of all create and destroy events of objects.

 

G# Version

Version of G#.

 

Application Instance

Application instance of the object scope.

 

Owner

Owning top-level VI of the selected object. When this VI stops executing, LabVIEW will automatically garbage collect the object.

 

Filter Expression

Filters the trace log and only showing lines containing the expression.
