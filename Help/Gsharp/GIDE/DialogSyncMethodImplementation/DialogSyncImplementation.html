Dialog - Synchronize Method Implementation

Synchronizes method implementations so that the method signature (connector pane) matches for all implementations of the method.

 Sync

Synchronizes according to the chosen settings

Cancel

Cancel.

Help

Help.

Affected implementations. Select the method to use as source.

The affected implementations of the specified method that will be updated. The selected implementation will be used as source and all affected implementations of the method will update their connector panes to match the selected implementation. Controls that are present on the same connector pane terminal on both the source and the destination and have the same type will only update documentation and default value, if those options are selected in the "Synchronization Options" of this dialog. Controls that do not match the type for the source and destination or do not exist at the destination connector pane terminal will be duplicated from the source implementation. Controls connected to the connector pane which does not match the selected implementation will be removed.

Double-click to open a VI.

Synchronization Options

Documentation

If selected, any control that already exists in the target implementation (VI) with the same type will have its documentation and tip-strip replaced with the documentation of the corresponding control in this method.

Default Value

If selected, any control that already exists in the target implementation (VI) with the same type will have its default value replaced with the default value of the corresponding control in this method.
