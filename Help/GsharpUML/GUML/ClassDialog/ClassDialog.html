Dialog - Select Classes to Create or Update

This dialog specifies where to create the G# classes from the UML diagram.

 Class name

Name of the class.

Class template

Template to use when creating the class.

Class path

Path to the class. A folder called <ClassName>_class will be created inside this folder containing the actual class.

Header color

Class icon header color.

Body color

Class icon body color.

Create or update class

Controls if the class should be create,updated or ignored.

Help

Launces help.
