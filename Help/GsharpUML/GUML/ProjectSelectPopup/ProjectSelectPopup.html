Dialog - Select Project

This dialog chooses the project to use as a source of operation.

 Collapse Operations

Controls if the operations should be presented collapsed or not in the UML diagram (for readability).

Collapse Attributes

Controls if the attributes should be presented collapsed or not in the UML diagram (for readability).

Include Dependencies

Controls if dependencies should be included in the analysis of classes. Inheritance is always included dispite the setting of this control.

Include G#Object

Controls if the G#Object should be included in the analysis or not. If enabled the UML diagram could be difficult to read.

Help

Launces help.
